## Chiffrement en Morse

Ce code a pour but de chiffrer et déchiffrer des textes en Morse.

Dans les textes, les caractères autorisés sont :
- les lettres de l'alphabet
- les chiffres
- les signes de ponctuations suivants : . , ? ' ! / ( ) & : ; = + - _ " @
